# Voting System

## Project Description
This is a full-stack voting system web application that allows users to securely vote and view results in real-time. The system ensures proper data handling and user authentication.

## Features
- User registration and login system  
- Secure voting functionality  
- Real-time vote counting  
- Database integration  
- Simple and user-friendly interface  

## Technologies Used
- Backend: Python (Flask)  
- Frontend: HTML, CSS  
- Database: MySQL / SQLite  
- Tools: Git, GitHub
